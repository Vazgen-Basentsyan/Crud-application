
Install required system packages:

.. code-block:: bash
```
$ sudo apt-get install python3.6-dev
```
```
$ sudo apt install pipenv
```

```
$ pipenv isntall
```
For activate virtual environment 

```
$ pipenv shell
```
For run 
```
$ python manage.py migrate
$ python manage.py runserver
```
